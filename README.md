## 🚀 Featured Projects

### 🌾 AgroSmart — AI-Powered Smart Irrigation System
> ESP32 · Embedded C · DHT11 · HC-SR04 · Soil Moisture · Relay · OpenWeatherMap API · Telegram Bot · REST API

Built end-to-end IoT firmware for an ESP32-based irrigation system as part of a 4-person team for **Smart India Hackathon 2025** and two college hackathons.

**What it does:**
- Reads DHT11 (temp/humidity), soil moisture sensor, HC-SR04 ultrasonic water level
- Controls a 12V pump via relay with automatic safety cut-offs
- Serves a live web dashboard (Chart.js 24-hr sensor graphs) via on-device HTTP server
- Telegram Bot with 10+ commands (`/pumpon`, `/pumpoff`, `/auto`, `/sensors`, `/ai`)
- Integrates OpenWeatherMap API — pauses irrigation automatically during forecasted rain
- AI water-budgeting algorithm based on crop type, soil profile, and climate data
- Flash (NVS) storage for persistent farmer profile across power cycles

### 📟 ARM Cortex-M3 Peripheral Driver Library
> LPC1768 · ARM Cortex-M3 · Embedded C · Keil µVision4 · Register-level · No HAL

**12 bare-metal peripheral drivers** for the NXP LPC1768 microcontroller — written directly against hardware registers without HAL or CMSIS abstraction.

**Modules implemented:**
`GPIO` · `LCD (8-bit)` · `4×4 Keypad` · `ADC (LM35)` · `Timer0/Timer1` · `PWM` · `RTC` · `UART0` · `SPI0 Master` · `I2C0 Master` · `External Interrupts (EINT0/EINT1)` · `PLL Clock Config`

Each module includes pinout reference, register-level explanation, and build/flash guide (Keil µVision4 + Flash Magic).
